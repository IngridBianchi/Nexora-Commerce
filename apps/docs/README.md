# Docs App - Nexora Commerce

Aplicacion de documentacion del monorepo, basada en Next.js y preparada para evolucionar guias tecnicas y funcionales del proyecto.

## Stack

- Next.js 16
- React 19
- TypeScript
- Configuraciones compartidas de eslint y tsconfig

## Scripts

```bash
npm run dev
npm run build
npm run start
npm run lint
npm run check-types
```

## Desarrollo local

Esta app usa el puerto 3002 por defecto.

```bash
npm run dev
```

Abrir http://localhost:3002

## Validacion de calidad

```bash
npm run lint
npm run check-types
```
