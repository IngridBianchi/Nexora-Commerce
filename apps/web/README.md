# Web App - Nexora Commerce

Aplicacion secundaria Next.js del monorepo, orientada a experiencias web adicionales y consumo de componentes compartidos.

## Stack

- Next.js 16
- React 19
- TypeScript
- Paquete compartido @repo/ui

## Scripts

```bash
npm run dev
npm run build
npm run start
npm run lint
npm run check-types
```

## Desarrollo local

Esta app usa el puerto 3001 por defecto.

```bash
npm run dev
```

Abrir http://localhost:3001

## Calidad

```bash
npm run lint
npm run check-types
```
